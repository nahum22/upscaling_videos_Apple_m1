# Model Weights

Place the Real-ESRGAN model weights here before running the worker.

## Download RealESRGAN_x4plus.pth

```bash
curl -L -o RealESRGAN_x4plus.pth https://github.com/xinntao/Real-ESRGAN/releases/download/v0.1.0/RealESRGAN_x4plus.pth
```

Or download manually from:
https://github.com/xinntao/Real-ESRGAN/releases/download/v0.1.0/RealESRGAN_x4plus.pth

The worker expects the file at:
`worker/weights/RealESRGAN_x4plus.pth`
